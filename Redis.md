# Redis

类型：

使用内存存储的非关系型数据库

数据类型：

字符串、列表、集合、散列表、有序集合

查询类型：

每种数据类型有专属的命令，还有批量操作和不完全的事务支持



主从复制、持久化、脚本（存储过程）





| 结构类型 | 结构存储的值 | 结构的读写能力 |
| -------- | ------------ | -------------- |
|          |              |                |

`redis-cli`	进入redis交互式客户端

## 字符串

SET	`set hello world`	==>	`pk`

GET	`get hello`	==>	`'world'`	

DEL	`del hello`	==>	`(integer) 1`

## 列表

`LPUSH`	将元素插到列表的左端

`RPUSH`	将元素插到列表的右端

`LPOP`	   从列表的左端弹出元素		

`RPOP`	   从列表的右端弹出元素

`LINDEX`	获取列表在给定位置上的一个元素                 索引从0开始

`LRANGE`	获取在给定范围上的所有元素

## 集合

`SADD`	将元素添加到集合

`SREM`	删除元素

`SISMEMBER`	检查一个元素是否存在于集合中         member：成员

`SMEMBERS`	获取集合的所有元素

## 散列（存储键值对）

`HSET`	设置键值对

`HGET`	获取指定的键值对

`HGETALL`	获取指定key下的所有键值对

`HDEL`	删除指定的键值对

## 有序集合（存储键值对）

键称为成员（member）  值称为分值（score）

`ZADD`	将带有分值的成员添加到有序集合里

`ZRANGE`	根据元素位置获取多个元素

`ZRANGEBYSCORE`	获取给定分值范围内的所有元素

`ZREM`	删除成员
